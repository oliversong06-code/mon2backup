# AssetFlow 전체 자산 목업

PC용 자산관리 서비스의 2페이지 `전체 자산` 화면입니다. 1페이지와 같은 다크 네이비 테마, 사이드바, 카드 스타일, 색상 토큰을 사용한 정적 HTML 목업입니다.

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열면 됩니다.

## 구성

- 페이지 헤더와 `자산 구성해보기` 정적 버튼
- 자산 구성에서 확인된 특징 카드
- 자산 구성 도넛 차트와 보유 자산 내부 스크롤 목록
- 가장 먼저 확인할 구성 스택바
- 주요 영향 요인 독립 수평 바
- 최근 1개월 수익 기여도 절대금액 표시

점수, 평가 게이지, 단정적 위험 표현은 사용하지 않았습니다.
