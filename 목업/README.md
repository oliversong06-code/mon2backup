# AssetFlow PC 자산관리 대시보드 목업

정적 HTML/CSS/JavaScript로 만든 PC 전용 첫 화면 목업입니다. 실제 로그인, API 호출, 저장, 페이지 이동 기능은 포함하지 않았고, 버튼과 메뉴는 hover 상태만 확인할 수 있습니다.

## 실행 방법

1. `index.html` 파일을 브라우저에서 직접 엽니다.
2. 인터넷 연결이 있으면 Pretendard 폰트와 Chart.js CDN이 로드됩니다.
3. CDN이 차단된 환경에서도 레이아웃은 보이며, 차트 영역만 정상 렌더링되지 않을 수 있습니다.

## 파일 구성

```text
asset-dashboard-mockup/
├─ index.html
└─ README.md
```

## 구현 요약

- 240px 고정 사이드바와 12컬럼 메인 그리드 구조
- 총자산, 투자자산, 안전자산, 이번 달 변동 요약 카드
- Chart.js 기반 자산 구성 도넛 차트와 6개월 자산 추이 라인 차트
- 포트폴리오 진단, 확인 필요 항목, 보유 자산 테이블
- 향후 기능 카드 3종을 낮은 강조도로 배치
- 접근성을 위한 aria-label, table 시맨틱 구조, 충분한 명도 대비 적용
