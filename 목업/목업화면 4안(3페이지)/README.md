# 목업화면 4안(3페이지)

해외주식 자산군 상세 화면의 네 번째 비교안입니다.

- 기존 3페이지 최신안은 유지하고 별도 폴더로 분기했습니다.
- 상단 분석 카드를 구성 특성, 가격 요인, 환율 요인을 예고하는 미리보기 카드로 개편했습니다.
- 최근 1개월 손익분해는 0 기준선을 중심으로 양수와 음수를 좌우 방향으로 표현하는 발산형 막대로 변경했습니다.
- 자산군 선택은 전체 자산, 현금·예금, 국내주식, 해외주식, ETF, 연금, 부동산, 가상자산, 기타 순서로 정리했습니다.
- 현재 구현된 이동은 전체 자산과 해외주식만 제공하고, 나머지는 준비 중 비활성 상태로 표시합니다.
